# Text Summarizer Project Plan

## Project Title
Text Summarizer using NLP (NLTK)

## Problem Statement
Reading long articles takes time. This project will generate a short summary from a long text automatically.

## Objectives
- Accept input text
- Preprocess text
- Remove stop words
- Score sentences
- Generate summary
- Display original text and summary

## Technologies Used
- Python
- NLTK
- VS Code
- Git
- GitHub

## Future Enhancements
- Upload text files
- Web interface
- Better summarization algorithms
