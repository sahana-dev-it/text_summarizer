# NLP Text Summarizer

## Project Description

This project is a Text Summarizer developed using Python, NLTK, and Tkinter. It allows users to enter large text and generate a concise summary using Natural Language Processing techniques.

## Features

* User-friendly GUI using Tkinter
* Enter custom text for summarization
* Short, Medium, and Long summary options
* Clear input and output fields
* Save generated summaries
* Uses NLTK for tokenization and stopword removal

## Technologies Used

* Python
* Tkinter
* NLTK
* GitHub

## How to Run

```bash
python src/gui_summerizer.py
```
