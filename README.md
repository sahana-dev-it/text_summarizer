# AI Text Summarizer

## Author
Sahana GS

## Problem Statement
Reading long news articles takes time. This project automatically generates a shorter summary by identifying the most important sentences from a news article.

## Dataset Source
BBC News Summary Dataset

Dataset Categories:
- Business
- Entertainment
- Politics
- Sport
- Tech

Total Articles: 2225

## Tech Stack
- Python
- NLTK
- Git
- GitHub
- VS Code

## Project Approach

1. Read article from dataset
2. Tokenize text into sentences and words
3. Remove stopwords
4. Calculate word frequencies
5. Score sentences
6. Select highest scoring sentences
7. Generate summary

## Features

- Summarizes BBC news articles
- User selects category
- User selects article
- File error handling
- Automatic summary generation
